# dwEvent

### Sobre o sistema

Sistema de gerenciamento de eventos. Que recebe trabalhos, avalia, confirma pagamentos e etc. Não é um portal mas apenas um painel.

### Pendências
- Sistema de login
- CRUD
- Sistema de envio de arquivos
- Sistem de avaliação de trabalhos
